--- ### @mixin fluid-type Fluid font-size between breakpoints #### Arguments | Name | Type | Default | Description | | ---- | ---- | ------- | ----------- | | `min-value` | `string` | - | The minimum font-size | | `max-value` | `string` | - | The maximum font-size | | `min-screen` | `string` | `$beta` | Breakpoint for fluid-type to kick in | | `max-screen` | `string` | `$delta` | Breakpoint for fluid-type to stop | #### SCSS {% highlight css %} .foo { @include fluid-type(16px, 24px); } {% endhighlight %} #### SCSS {% highlight css %} .foo { font-size: 16px; } @media screen and (min-width: 560px) { .foo { font-size: calc(1.724137931vw + 6.3448275862px); } } @media screen and (min-width: 1024px) { .foo { font-size: 24px; } } {% endhighlight %}
