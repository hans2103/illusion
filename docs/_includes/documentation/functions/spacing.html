### @function spacing() Returns set spacing or calculate current spacing, also within breakpoint #### Arguments | Name | Type | Default | Description | | ---- | ---- | ------- | ----------- | | `multiplier` | `number` | `1` | Gutter multiplier. defaults to `1` and can be set to any number dividable by `0.5` | | `breakpoint` | `string` | `false` | Specific breakpoint to use | #### SCSS {% highlight css %} .foo { @include breakpoint($bravo) { padding-top: spacing(); } } {% endhighlight %} #### CSS {% highlight css %} @media screen and (min-width: 560px) { .foo { padding-top: 24px; } } {% endhighlight %}
