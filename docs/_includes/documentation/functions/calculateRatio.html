--- ### @function calculateRatio() Calculates a ratio between two numbers and returns it as percentage. Calculated by using the width of the element. The output will not work for flex items. #### Arguments | Name | Type | Default | Description | | ---- | ---- | ------- | ----------- | | `ratio` | `list` | - | Returns ratio as percentage. Accepts a list like `16 by 9` | #### SCSS {% highlight css %} .foo { padding-top: calculateRatio(16 by 9); } {% endhighlight %} #### CSS {% highlight css %} .foo { padding-top: 56.25%; } {% endhighlight %}
