--- ### @function getWidth() Returns width, also within breakpoint #### Arguments | Name | Type | Default | Description | | ---- | ---- | ------- | ----------- | | `span` | `list` | - | Set how many columns should be spanned. Accepts single value: `getWidth(6)` or list: `getWidth(4 of 12)`. Amount of columns defaults to `$illusion-grid-container` | | `multiplier` | `number` | `1` | Gutter multiplier. defaults to `1` and can be set to any number dividable by `0.5` | | `gutter` | `string` | `false` | Custom gutter value for Illusion to calculate (`calc()`) with. | #### SCSS {% highlight css %} .foo { @include breakpoint($bravo) { width: getWidth(6); } } {% endhighlight %} #### CSS {% highlight css %} @media screen and (min-width: 560px) { .foo { width: calc((100% - 24px) / 2); } } {% endhighlight %}
